# Bank Management System

## Overview

The Bank Management System is a console-based application developed in C++ using Object-Oriented Programming (OOP) concepts. This project allows users to manage bank accounts efficiently through a menu-driven interface. It provides essential banking operations such as account creation, searching, updating, deleting, depositing, withdrawing, and transferring money between accounts.

This project was created as a learning exercise to strengthen understanding of classes, objects, constructors, vectors, encapsulation, and basic banking operations in C++.

---

## Features

* Create New Account
* Display All Accounts
* Search Account by Account Number
* Update Account Information
* Delete Account
* Deposit Money
* Withdraw Money
* Transfer Money Between Accounts
* Password Masking During Display
* Validation for Invalid Transfer Amounts
* Validation for Same Sender and Receiver Accounts

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/MuhammadDuryabBakht/BankManagementSystem.git
```

2. Open the project in any C++ IDE or compiler.

3. Compile the program:

```bash
g++ main.cpp -o bank
```

4. Run the executable:

```bash
./bank
```
